CSS Exercise 10

Styling hyperlinks in text and menus

Focus on:

  • Using dynamic pseudo-class selector to style links according to user interaction
  • Understanding how descendant selectors allow you to isolate ubiquitous tags and only style the instances you want according to those tags' ancestry in the document tree
  • Reminders about using background images. There are two. The screenshot will make clear what they are and where they go.

Overview

Use the screen shot to help guide to to style the hyperlinks in the page. Replicate the screen shot as closely as you can. The layout styling is done already so you can focus on the hyperlink styling.

There are hyperlinks in these three regions of the page. Descendant selectors allow you to isolate them all so the styles of the hyperlinks can be different:

  • General styling of hyperlinks (so nothing is left out)
  • Site ID in the banner
  • Submenus in the sidebar

You have two background images to put into the design. The screenshot should make it clear which one goes where in the layout.

Workflow

Most of the page styling is done already.

  1. Begin by putting in the background images. Pay attention to the screenshot!

  2. Style the general hyperlink styles first:

    • color of text hyperlinks is darkblue
    • visited link color of text hyperlinks is black
    • hover color of text hyperlinks is blue
    • active color of text hyperlinks is black
    • be sure you consult the screen shot to see the specific details of hover

  3. Style the Site ID second:

    • color of Site ID is black
    • hover color of Site ID is maroon
    • active color of Site ID is white

    You'll need to write a very specific selector to style the site id. My in-class demo shows how this is done. This is because of the cascade.

  4. Style the sidebar submenus last:

    • color of sidebar hyperlinks is maroon
    • hover color of sidebar hyperlinks is red
    • active color of sidebar hyperlinks is black
    • be sure you consult the screen shot to see the specific details of hover, and the spacing between hyperlinks, and between menus and their heading.
    • a lot of default styling of lists has been changed!

Validation

You want to be sure you've written valid markup and valid styles. Use the online validator tools to check your work. If there are errors, read what the validator says about them and check the lines in your markup where the errors occur.
